파이프 오프셋 계산

각도와 높이차(Set)를 입력하면 Travel과 Run을 계산합니다.

공식 보기

Travel = Set / sin(θ)

Run = Set / tan(θ)

45° 오프셋: Travel = Set × 1.414, Run = Set

엘보 계산

기성품 엘보 테이크아웃

테이크아웃 = 중심~끝 거리. 도면 C-C 치수에서 양쪽 테이크아웃을 빼면 파이프 절단 길이.

탄젠트 엘보 (직관 제작)

직관 파이프를 마이터 절단하여 엘보를 제작할 때 절단 각도와 마킹 치수를 계산합니다.

공식 보기

기성품 90° LR: 테이크아웃 = NPS × 1.5

기성품 90° SR: 테이크아웃 = NPS × 1.0

기성품 45° LR: 테이크아웃 = NPS × 0.625

탄젠트 절단 각도 = 회전각도 / (2 × 용접수)

파이프 마킹 = OD × tan(절단각도)

2-piece 90°: 45° 절단, 마킹 = OD. 3-piece 90°: 22.5° 절단, 마킹 = OD × 0.414

튜빙 절단 길이

도면 치수에서 피팅 허용량과 용접 수축을 빼고 실제 절단 길이를 계산합니다.

mm
공식 보기

절단 길이 = 전체 길이 - 피팅 허용량 - (수축량 × 용접 수)

수축량은 단위에 주의. 전체 길이가 inch면 수축량도 inch로 변환됩니다.

파이프 원주

외경(OD)을 입력하면 원주와 반원주를 계산합니다. 마킹/래핑 템플릿용.

공식 보기

원주 = π × OD

반원주 = π × OD / 2

용접 수축량

파이프 사이즈와 용접 개소 수를 입력하면 총 수축량을 계산합니다.

공식 보기

총 수축량 = 1개소 수축량 × 용접 개소 수

고순도 SST TIG 기준. 실제 값은 전류/패스 수/갭에 따라 달라짐.

단위 변환

inch ↔ mm 상호 변환. 분수 inch도 지원합니다.

자주 쓰는 분수 inch

공식 보기

mm = inch × 25.4

inch = mm / 25.4

각도 / 투영 보정

도면 치수와 각도를 입력하면 실제 파이프 컷팅 길이를 계산합니다. 각도 편차를 추가 입력하면 오차 영향도 함께 분석합니다.

°
공식 보기

투영 보정: 실제 파이프 = 도면치수 / cos(θ) (수평 투영일 때)

높이 = 파이프길이 × sin(θ)

수평거리 = 파이프길이 × cos(θ)

높이 변화 = 길이 × |sin(θ±δ) - sin(θ)| (편차 입력 시)

6°에서 2.30" → 차이 0.013" (직선 OK). 45°에서 12.97" → 직선으로 자르면 5.37" 차이!

용접 파라미터 (펄스 TIG)

펄스 TIG 설정값으로 열입력을 계산하고, STS 열변형을 최소화하는 턴테이블 RPM을 추천합니다.

공식 보기

BG 전류 = Peak × BG%

평균 전류 = Peak × Duty% + BG전류 × (100-Duty%)

이동 속도 = RPM × π × OD (mm/min)

열입력 = (V × I_avg × 60) / Speed (J/mm)

리플 간격 = 이동속도(mm/s) / PPS

STS 열변형 기준: 0.5 kJ/mm 이하 권장. 1.0 kJ/mm 초과 시 변형 위험. Duty 낮추고 RPM 올리면 열입력 감소.

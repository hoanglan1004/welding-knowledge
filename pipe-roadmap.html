1단계 · 제련

⛏️ 철광석에서 스테인리스강까지

돌멩이(철광석)를 녹여서 깨끗한 철(스테인리스강)로 만드는 과정

탄소강
🏗️
경로 A: 고로(BF) → 전로(BOF)
철광석 + 코크스를 1600도에서 녹여 선철을 만들고,
전로에서 산소를 불어넣어 탄소를 태워 없앱니다.
🌡️ 1600~1700℃

결과물: 건물, 다리에 쓰는 일반 철강

스테인리스
⚗️
경로 B: 전기로(EAF) → AOD
폐철(스크랩) + 크롬 + 니켈을 전기 아크로 녹이고,
AOD에서 아르곤 가스로 불순물을 날려보냅니다.
⚡ 1600~1800℃ (전기 아크)

결과물: 녹슬지 않는 스테인리스강 (304L, 316L)

요리에 비유하면, 경로 A는 원래 재료(철광석)로 처음부터 요리하는 것이고,
경로 B는 남은 음식(스크랩)에 양념(크롬, 니켈)을 넣어 고급 요리로 만드는 것!

1단계 · AOD 공정

🫧 AOD: 스테인리스의 비밀 무기

아르곤-산소 탈탄 — 탄소만 쏙 빼고 크롬은 지키는 마법

AOD는 국 끓일 때 거품 걷어내는 것과 비슷해요.
산소로 나쁜 것(탄소)을 태워 없애고, 아르곤으로 좋은 것(크롬)은 다시 건져올리는 거예요.
탄소가 0.03% 이하로 내려가야 "L급" (Low carbon) 스테인리스가 됩니다!

1단계 · 주조

🧊 연속주조: 쇳물을 블록으로

뜨거운 쇳물을 틀에 부어 파이프의 원재료(빌렛)를 만드는 과정

1

🫗 쇳물 붓기 (Tundish)

정련 완료된 1600도 쇳물을 중간 그릇(턴디시)에 부어요

2

🧊 틀에 넣기 (Mold)

사각형 틀(주형)에 흘려보내면 바깥부터 굳기 시작!

3

💦 물 뿌려 식히기

물을 뿌려서 빠르게 굳혀요. 속도가 너무 빠르면 결함이 생겨요!

4

✂️ 잘라서 빌렛 완성

120mm × 120mm 사각 블록으로 절단 → 빌렛(Billet) 완성!

아이스크림 만들기와 비슷해요! 액체(쇳물)를 틀에 붓고 → 식히면 → 단단한 블록이 되는 거예요.
이 블록(빌렛)이 파이프의 원재료가 됩니다.

2단계 · 파이프 성형

🔧 빌렛을 파이프로 만드는 두 가지 방법

🔥
1. 빌렛 가열
1000~1200도로 달궈서 찰흙처럼 물렁하게 만들어요
🕳️
2. 천공 (뚫기)
뜨거운 빌렛을 회전시키면서 가운데를 뚫어요! (Mannesmann 공법)
📏
3. 신축 (늘리기)
롤러로 밀어서 원하는 두께와 길이로 늘려요

찰흙 놀이와 같아요! 찰흙 덩어리(빌렛)에 나무젓가락을 꽂아서 구멍을 뚫고,
양손으로 굴려서 얇고 길게 만드는 거예요. 이음매가 없어서 100% 강도!

📜
1. 코일 풀기
얇은 철판을 두루마리 휴지처럼 쭉 풀어요
🔄
2. 둥글게 말기
롤러 사이를 통과하면서 U자 → O자로 둥글어져요
⚡
3. 전기로 접합
가장자리에 고주파 전류를 흘려 1300도로 녹여 붙여요!

종이로 빨대 만들기와 같아요! 종이(철판)를 둥글게 말고 → 풀로 이음새를 붙이는 거예요.
다만 이음새(용접선)가 하나 있어서 심리스보다 강도가 살짝 낮아요 (90~95%).

2단계 · 심리스 파이프

심리스 파이프 제조 6단계

유리 불기(Glass Blowing)와 비슷해요!
뜨거운 유리를 돌리면서 불어서 빈 공간을 만들고 → 늘리고 → 식히면 → 파이프 완성!

BA 튜브는 일반 파이프의 "프리미엄 버전"이에요. 반도체 공장처럼 먼지 하나도 안 되는 곳에 쓰여요.
피팅은 수도관의 이음쇠와 같아요 — 파이프끼리 연결하거나 방향을 바꿀 때 필요!

4단계 · 표면처리

✨ 파이프를 반짝반짝하게!

더러운 표면을 깨끗하고 녹슬지 않게 만드는 과정

1

🧪 산세 (Pickling) — 때 벗기기

산성 용액(염산+불산)에 담가서 검은 산화층(때)을 녹여 없애요.
⚠️ 불산은 매우 위험! 전문가만 취급

2

🛡️ 패시베이션 — 보호막 입히기

질산 용액에 담가서 크롬이 표면에 얇은 보호막을 만들게 해요.
이 막이 녹(부식)을 막아주는 스테인리스의 비밀!

3

⚡ 전해연마 (EP) — 거울처럼 만들기 [클릭하면 자세히]

전기를 흘려서 울퉁불퉁한 표면의 튀어나온 부분을 녹여 평탄하게!
반도체용 파이프는 이 단계가 필수예요.

⚡ 전해연마(EP)는 어떻게 작동할까?

파이프를 전해액에 넣고 전기를 흘리면, 튀어나온 부분이 먼저 녹아요 (전류가 집중되니까!).
들어간 부분은 그대로 남아서, 결과적으로 울퉁불퉁 → 매끈매끈이 됩니다.

산(울퉁불퉁) → 평원(매끈) 으로 바뀌는 거예요!
결과: 표면 거칠기 Ra ≤ 0.25μm, Cr/Fe 비율 증가 → 부식 저항 극대화

세차와 같아요! 산세 = 고압 세척으로 진흙 제거, 패시베이션 = 왁스 코팅, EP = 광택 폴리싱!

5단계 · 품질인증

📋 MTR: 파이프의 신분증

이 파이프가 진짜 좋은 건지 증명하는 서류

🧬
화학 성분 검사
파이프에 뭐가 들어있는지 확인!
C (탄소): ≤ 0.030% ✅
Cr (크롬): 16~18% ✅
Ni (니켈): 10~14% ✅
Mo (몰리): 2~3% ✅
💪
기계적 성질 검사
얼마나 튼튼한지 테스트!
🔨 항복강도: ≥170 MPa
💥 인장강도: 485~620 MPa
📐 연신율: ≥40%
🧊 충격시험: ≥40J
🔍
추적성 (Heat No.)
모든 파이프에 고유번호!
🏭 어디서 만들었는지
📅 언제 만들었는지
🧪 무슨 재료인지
✅ 검사 통과했는지

문제 생기면 같은 번호 전부 회수 가능!

MTR은 사람의 건강검진 결과표와 같아요!
혈액검사(화학성분) + 체력검사(기계적 성질) + 주민번호(Heat Number)가 모두 적혀있어요.
이 서류가 없으면 그 파이프는 공사현장에서 쓸 수 없어요!

FULL ROADMAP

전체 여정: 광석에서 파이프까지

⛏️ 1단계 제련 (1600~1800℃)

철광석/스크랩 → EAF(전기로) → AOD(정련) → 연속주조 → 빌렛

🔧 2단계 파이프 성형

빌렛 → 가열 → 천공/말기 → 신축 → 냉각 → 파이프 (심리스 or ERW)

🔩 3단계 피팅 제조 + BA 튜브

단조/압출로 엘보·티·리듀서 제작 | 냉간인발+BA열처리 → 반도체급 튜브

✨ 4단계 표면처리

산세(때 제거) → 패시베이션(보호막) → EP(전해연마) → 거울급 표면

📋 5단계 품질인증 (MTR)

화학성분 + 기계적 성질 + Heat Number → MTR 발급 → 출하!

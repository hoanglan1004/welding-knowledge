파이프 계산기 사용법

총 11개 계산기가 탭으로 구성되어 있습니다. 상단 탭을 눌러 필요한 계산기를 선택하세요.
아래 각 항목을 눌러 사용법을 확인하세요.

1. 공학 계산 기본

▼

삼각함수, 제곱근 등이 내장된 현장용 공학 계산기입니다.

버튼 설명

sin cos tan 삼각함수 (기본: DEG 모드)
√ x² π ( ) 루트, 제곱, 파이, 괄호
DEG 누를 때마다 DEG ↔ RAD 전환

사용 예시

1
sin(45) 계산하기
sin → 4 → 5 → =
결과: 0.7071

괄호는 = 누르면 자동으로 닫힙니다! 직접 닫을 필요 없어요.

2
√(144) 계산하기
√ → 1 → 4 → 4 → =
결과: 12
3
원주 계산: π × 2 (OD 2인치 파이프 원주)
π → × → 2 → =
결과: 6.2832 (inch)
팁: AC는 전체 초기화, DEL은 마지막 글자만 삭제.

용접 공식 (아래쪽 섹션)

공식 입력 결과
펄스 전류 Peak(A), BG(%), Duty(%) RMS 전류, 산술평균, BG 전류
열입력 전압(V), 평균전류(A), 속도(mm/min) kJ/mm + 판정 (최적/주의/위험)
속도 변환 IPM 또는 RPM + OD(inch) IPM ↔ RPM ↔ mm/min

2. 용접 레시피 핵심

▼

파이프 사이즈와 벽두께만 선택하면 펄스 TIG 세팅을 자동 생성합니다.

1
파이프 사이즈 선택 (예: 2" OD)
2
벽두께 선택 (자동으로 해당 사이즈 벽두께 목록 표시)
3
소재 선택 (304 BA / 316L BA)
4
용접 타입: 맞대기(Butt) 또는 플랜지(Fillet)
5
레시피 생성 버튼 클릭

결과로 나오는 것:

Peak / BG / Duty / PPSDynasty 300에 바로 입력할 값
RPM (최적 / 범위)턴테이블 속도
열입력kJ/mm + 판정
전극 / 용가재 / 가스준비물 한눈에
원리: 1A/mil(0.001") 업계 표준 기반. OD 열용량 보정 자동 적용. 플랜지는 히트싱크 보상(x1.18) + 높은 Duty(70%).

3. 백퍼지 핵심

▼

파이프 사이즈 + 벽두께 + 길이만 넣으면 유량, 시간, 필요 가스량을 자동 계산.

1
파이프 사이즈 선택
2
벽 두께 입력 (mm 또는 inch)
3
퍼지 구간 길이 입력 (끝~끝 거리)

결과로 나오는 것:

퍼지 시간산소 10ppm 도달까지 필요한 시간
Dwyer 플로미터 설정초기 퍼지 SCFH + 용접 중 SCFH
입/출 벤트 추천벤트 사이즈 + 위치 (하부 입구, 상부 출구)
산소 농도 변화표시간별 예상 O₂ ppm
중요: 계산값은 이론치입니다. 반드시 산소 분석기로 10ppm 이하 확인 후 용접 시작하세요.

4. 용접설정 (분석) 심화

▼

현재 쓰고 있는 설정값을 입력하면 업계 표준과 비교 분석해줍니다.

레시피가 "자동 추천"이라면, 용접설정은 "내 설정 진단"입니다.

1
용접 타입 선택 (맞대기 / 플랜지)
2
파이프 사이즈 + 벽 두께 입력
3
현재 사용 중인 Peak, BG%, PPS, Duty, 전압 입력
4
턴테이블 RPM 입력 (선택 - 넣으면 열입력도 분석)

결과: 내 설정 vs 추천 비교표 + OK/높음/낮음 판정 + 조정 가이드

5. 파이프 오프셋

▼

파이프 높이를 변경할 때 Travel(사선 길이)과 Run(수평 거리)을 계산.

1
각도 선택 (45도가 가장 일반적)
2
Set (높이차) 입력
45도, Set 6" → Travel = 8.485", Run = 6"
공식: Travel = Set / sin(θ), Run = Set / tan(θ)
45도 오프셋이면 Travel = Set × 1.414, Run = Set (같음)

6. 엘보

▼

두 가지 기능이 있습니다:

A. 기성품 엘보 테이크아웃

파이프 사이즈 + 엘보 타입(90LR/90SR/45LR) 선택 → 테이크아웃(중심~끝 거리) 표시

도면의 C-C 치수에서 양쪽 테이크아웃을 빼면 = 파이프 절단 길이

B. 탄젠트 엘보 (직관 제작)

직관 파이프로 엘보를 만들 때 절단 각도와 마킹 치수를 계산.

파이프 사이즈 + 회전 각도 + 피스 수 선택 → 절단 각도 + 마킹 거리

7. 절단길이

▼

도면 치수에서 피팅 허용량과 용접 수축을 빼서 실제 절단 길이를 계산.

도면 전체 = 24" - 피팅 허용량 3" - 수축(2mm × 2개소) = 20.843"
수축량은 mm로 입력합니다. 전체 길이가 inch면 자동 변환됩니다.

8. 수축량

▼

파이프 사이즈 + 용접 개소 수 → 총 수축량 (mm / inch)

고순도 SST TIG 기준 데이터 사용. 실제 값은 전류/패스 수에 따라 달라질 수 있습니다.

9. 원주

▼

외경(OD) 입력 → 원주(π × OD)와 반원주 계산. 마킹/래핑 템플릿에 활용.

10. 단위변환

▼

inch ↔ mm 실시간 변환. 숫자를 입력하면 즉시 반대쪽이 변환됩니다.

하단에 자주 쓰는 분수 inch 버튼이 있어요. 클릭하면 바로 변환됩니다.

예: 1/8 클릭 → 0.125" = 3.175mm

11. 각도/투영 보정 심화

▼

도면 치수가 투영(수평/수직)인 경우, 실제 파이프 길이를 계산합니다.

각도 편차를 추가 입력하면 허용 오차 범위도 분석합니다.

1
도면 치수(길이)와 각도 입력
2
(선택) 각도 편차 입력 → 오차 분석 추가
45도, 2.30" → 높이: 1.626", 수평: 1.626"
편차 ±1도 입력 → 높이 차이 0.028" (0.03" 이내 OK)

빠른 참조: 언제 뭘 쓰나?

상황사용할 탭
새 파이프 용접 시작할 때용접 레시피 (자동 세팅)
퍼지 준비할 때퍼지 (유량 + 시간)
현재 세팅 맞는지 확인할 때용접설정 (분석)
파이프 자를 때절단길이 + 수축량
배관 경로 설계할 때오프셋 + 엘보
도면 치수 변환할 때단위변환 + 원주
기울어진 배관 길이 계산각도/투영
현장에서 빠른 계산공학 계산
